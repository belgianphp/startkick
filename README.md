# BelgianPHP Start Kick
Folder suggestion

## Installation
```
$ git clone https://github.com/belgianphp/startkick.git dir_project
$ cd dir_project
$ composer install
```
or:
## Installation 2:
```
$ composer create-project belgian/startkick dir_project 
```
## In the browser: 
```
localhost/dir_project/web
```
